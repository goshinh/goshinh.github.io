## 编程练习
6.	参考代码如下：
```cpp
#include<iostream>
using namespace std;
int main()
{
	int m,n,sum;
	m=10;
	n=12;
	sum=m*n;
	cout<<"李叔叔送给敬老院"<<sum<<"个菜椒。" ; 
	return 0;
}
```
7.	参考代码如下：
```cpp
#include<iostream>
using namespace std;
int main()
{
	int m,n,sum;
```
